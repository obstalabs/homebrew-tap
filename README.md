# homebrew-tap

Homebrew formulae for Obsta Labs tools.

## Install

```bash
brew tap obstalabs/tap
brew install obstalabs/tap/neurorouter
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [neurorouter](https://github.com/obstalabs/neurorouter) | Community edition of NeuroRouter — cleans AI requests before they hit the model |

## Notes

- This tap currently publishes only public/community formulas.
- Formula updates are published from the source repository release workflow.
